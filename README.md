# ✨ Portfólio — Gustavo Steferson

Portfólio pessoal desenvolvido em **HTML, CSS e JavaScript puros**, com uma identidade visual em **preto e dourado**, precedido por uma página de verificação de acesso com efeito de mármore e ouro líquido animado.

---

## 🖤 Sobre o projeto

Este projeto é dividido em duas experiências:

1. **Página de verificação de acesso** (`index.html`) — um "portão" elegante que pede nome e e-mail antes de liberar o portfólio, com fundo de mármore animado (SVG + filtros de distorção líquida) e alternância entre tema claro e escuro.
2. **Portfólio** (`portfolio.html`) — página única apresentando formação, habilidades técnicas e contato, com uma trilha dourada lateral que acompanha a rolagem da página.

---

## 🚪 Página de verificação de acesso

Antes de chegar ao portfólio, o visitante vê uma tela que pede nome e e-mail. Ao confirmar, uma notificação é enviada automaticamente via **EmailJS**, e a pessoa é redirecionada para o portfólio.

**Destaques visuais:**
- 🌗 Alternância entre tema claro (mármore branco) e escuro (mármore preto), com preferência salva no navegador
- 🌊 Veios dourados animados com distorção líquida (SVG + `feTurbulence`), simulando ouro fluindo pelo mármore
- 🪞 Cartão com efeito de inclinação 3D que acompanha o cursor
- ✨ Brilho dourado que percorre a superfície do cartão continuamente

> ⚠️ Esta verificação não é uma proteção de segurança real — é apenas uma forma elegante de saber quem passou pelo link.
